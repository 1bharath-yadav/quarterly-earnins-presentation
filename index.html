// Calculate CAGR (annualized growth)
function cagr(beginValue, endValue, years) {
  return Math.pow(endValue / beginValue, 1 / years) - 1;
}

Financial Formula: CAGR

Compound Annual Growth Rate (CAGR):

\[ \mathrm{CAGR} = \left(\frac{V_f}{V_i}\right)^{\frac{1}{n}} - 1 \]
Explain variables: V_i = initial value, V_f = final value, n = years. Use when discussing growth assumptions.
